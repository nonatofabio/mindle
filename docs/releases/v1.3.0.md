## ✨ Auto-updates

Mindle now checks for and installs its own updates. Powered by [Sparkle](https://sparkle-project.org/) — the same framework Bear, Transmit, and most serious indie Mac apps use.

Every update is cryptographically verified with an EdDSA signature before it's allowed to run. Downloads come only from the signed, notarized DMGs published on this Releases page.

**Opt-in, off by default.** Mindle still won't reach out to any server unless you ask it to. On first launch after installing this version, Sparkle asks once whether to enable automatic checks — if you'd rather keep things quiet, just decline and nothing changes. You can always trigger a check manually from **Mindle → Check for Updates…**.

Thanks to [@agasthik](https://github.com/agasthik) for the nudge — closes [#1](https://github.com/nonatofabio/mindle/issues/1).
